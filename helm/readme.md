# Helm Charts

Helm charts for platform services and application packaging.

## Purpose

Reusable Helm charts for deploying services to the GKE cluster, consumed by Argo CD or deployed directly during platform setup.

## Planned Content

- Platform service charts (ingress controller, external-dns, cert-manager)
- Application chart templates
- Shared values and environment overrides

Work in progress — charts will be added in Phase 5 (Platform Services).
