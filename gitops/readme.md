# GitOps

Argo CD manifests and application definitions for Kubernetes workload management.

## Purpose

This directory will contain GitOps configuration using the App of Apps pattern to deploy and manage platform services and applications across environments.

## Planned Content

- Argo CD installation manifests
- App of Apps root application
- Environment-specific application overlays (dev, staging, prod)
- Platform service definitions (ingress, DNS, cert-manager, etc.)
